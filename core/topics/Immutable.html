Immutable Objects

When we modify a data of an object, if the result is stored in a new object but not in same object then it is called immutable object.


In Python int, float, bool, complex, string, tuple are immutable objects.


>>>x = 100
#print memory address
>>>print(id(x))
500961
>>>x = 200
#print memory address
>>>print(id(x))
4009782
x = 100 print(id(x)) x = 200 print(id(x))
